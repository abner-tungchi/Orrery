v0.2.2

解決 AI 多帳號的救星

為 Claude Code、Codex、Gemini CLI 隔離帳號，自由切換工作與個人情境 — 對話歷史無縫接續，不再從頭開始。

開始使用 GitHub
透過 Homebrew 安裝
$ brew install OffskyLab/orbital/orbital
或透過 APT 安裝（Ubuntu / Debian）
$ echo "deb [trusted=yes] https://offskylab.github.io/apt stable main" \
  | sudo tee /etc/apt/sources.list.d/orbital.list
$ sudo apt update && sudo apt install orbital
啟用 Shell 整合
$ source ~/.orbital/activate.sh

無痛切換所需的一切

🔒

憑證隔離

每個環境擁有獨立的 CLAUDE_CONFIG_DIR、CODEX_CONFIG_DIR、GEMINI_CONFIG_DIR — 認證 token 不會互相干擾。

💬

Session 共享

切換帳號後直接接續對話。Session 資料預設共享 — claude --resume 在切換環境後無縫運作。

🐚

Per-shell 生效

orbital use work 只在當前 shell 設定環境變數。其他終端機視窗維持各自的環境，互不影響。

✦

互動式精靈

透過方向鍵與空白鍵選擇工具，引導式建立環境 — 不需要記任何參數。

🔑

自訂環境變數

每個環境可儲存 API 金鑰等變數，orbital use 切換時自動匯出。

⚡

零執行開銷

Swift 原生編譯，啟動瞬間完成。Shell 整合只是一個輕量的 eval 函式，不是背景程式。支援 bash 與 zsh。

運作方式

1

安裝 Shell 整合

執行一次 eval "$(orbital setup)"。會自動將 shell function 寫入 rc 檔（~/.zshrc 或 ~/.bashrc，自動偵測）並立即生效。

2

建立環境

執行 orbital create work。互動式精靈讓你選擇工具並設定 session 共享。環境存放在 ~/.orbital/envs/。

3

一行指令切換

orbital use work 將正確的設定目錄與自訂環境變數匯出到當前 session。對話歷史在切換後完整保留。

4

接續上次的對話

切換帳號後，claude --resume 直接接續上一個 session。認證隔離，session 共享 — 兩全其美。

應用情境

💼

工作 / 個人帳號

公司用 Claude Team plan，自己有 Pro 訂閱。不用反覆登入登出，一行指令切換。

$ orbital use work
# → 使用公司 Team plan

$ orbital use personal
# → 使用個人 Pro plan
🏢

多客戶 / 多專案

接案者或顧問同時服務多個客戶，各自有不同的 API key 和憑證。一行指令切換情境。

$ orbital create client-a --tool claude
$ orbital create client-b --tool claude
$ orbital use client-a
💬

切換帳號不中斷對話

用工作帳號跟 Claude 對話到一半？切到個人帳號處理事情，再切回來 — session 還在原地等你。

$ orbital use work
$ claude
# ... 工作中 ...

$ orbital use personal
$ orbital use work
$ claude --resume
# → 接續上次的對話

指令一覽

環境管理
orbital create <name>建立新環境
orbital delete <name>刪除環境
orbital rename <old> <new>重新命名環境
orbital list列出所有環境
orbital info [name]顯示環境詳細資訊
切換
orbital use <name>在當前 shell 啟用環境
orbital deactivate停用目前的環境
orbital current顯示目前啟用的環境
orbital which <tool>顯示工具設定目錄路徑
設定
orbital tools [-e <name>]互動式管理工具
orbital set env <K> <V>設定環境變數
orbital unset env <K>移除環境變數
Session 管理
orbital sessions列出當前專案所有 session
orbital sessions --claude只看 Claude
orbital sessions --codex只看 Codex
orbital sessions --gemini只看 Gemini
Shell 整合
orbital setup安裝 shell 整合
orbital init輸出 shell function
